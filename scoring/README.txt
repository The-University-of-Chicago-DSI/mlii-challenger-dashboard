Scoring Python files

Usage from repo root:
python scoring/score_week_1.py
python scoring/score_week_8.py

Optional overrides:
python scoring/score_week_1.py --target-path hidden/Week_1_y_test.csv --submissions-dir submissions/week_1

Each script writes:
- week_X_scores.csv
- week_X_leaderboard_rows.csv
